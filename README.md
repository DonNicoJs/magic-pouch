# implementation-toolkit

> Frontend

## Build Setup

``` bash

# global CLI needed
$ brew install yarn
$ brew install node
$ yarn global add n
$ n latest
$ yarn global add ava

# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# test single run with coverage  
# open browsers on localhost:4000 to see coverage
$ yarn test


# build for production and launch server
$ yarn  build
$ yarn start

# generate static project
$ yarn run generate
```
